<p>How To Clone</p>
<p>1.Run git clone <Project Link></p>
<p>2.Run composer install</p>
<p>3.Run cp .env.example .env</p>
<p>4.Run php artisan key:generate</p>
<p>5.Run php artisan migrate</p>
<p>6.Run php artisan serve</p>
